# Super Store Sales Dashboard

## Project Overview

This project is an interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, and operational efficiency using the Super Store Sales Dataset. The dashboard provides actionable insights through dynamic visualizations and KPI tracking to support business decision-making.

## Key Insights

- Office Supplies generated the highest sales among all categories.
- Phones emerged as the top-performing sub-category.
- Consumer segment contributed the highest share of total sales.
- Standard Class was the most preferred shipping mode.
- Cash on Delivery (COD) accounted for the largest share of transactions.
- Sales and profit showed significant growth during the final quarter of the year.
